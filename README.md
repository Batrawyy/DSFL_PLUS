# DS-FL+

## Requirements
* Python version: `3.10.13`
* CUDA version: `11.6`

## Setup

```bash
git clone https://github.com/Kitsuya0828/DSFLplus.git
cd DSFLplus

conda env create -f env.yaml
conda activate dsflplus
```

## Run

```bash
cd src
python main.py --help
```
